# Paint
## Description
Projet d'éditeur graphique en C++/Qt similaire au logiciel Paint.
## Objectif
L'objectif principal de ce projet est de créer une application simple pour dessiner sur un écran. 
## Compilation et éxécution
``` 
$ qmake 
```
``` 
$ make 
```
``` 
$ ./paint 
```
